# Library Management System

Library Management System.

## Building

```bash
mkdir build && cd build
cmake ..
cmake --build .
```

## Running

```bash
./lms
```

## Testing

The project includes unit tests using GoogleTest:

```bash
./unit_tests
```

## Requirements

- CMake 3.20 or higher
- C++23 compatible compiler (GCC 13+, Clang 16+, MSVC 19.34+)
